# Security Policy

RDP Live Monitor is a defensive administration tool for Windows systems you own or are authorized to manage.

Please do not use it to interfere with systems without permission.

## Reporting Issues

If you find a bug or security issue, please open a GitHub issue with:

- Windows version
- PowerShell version
- Steps to reproduce
- Expected behavior
- Actual behavior

Avoid posting sensitive IP addresses, usernames, hostnames, or event logs publicly.
